Para compilar el programa, es necesario renombrar el fichero correspondiente a bmpBlackWhite.o, donde

bmpBlackWhite.o_LINUX32: es el fichero compilado para plataformas LINUX de 32 bits
bmpBlackWhite.o_LINUX64: es el fichero compilado para plataformas LINUX de 64 bits
bmpBlackWhite.o_MACOS: es el fichero compilado para plataformas MAC OSX
